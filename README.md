node-starter-kit
===========
06.19.2014

Node.js starter kit including express, based on [starter-kit][starterkit] and [Node.js][nodejs]

**Usage**
```
npm install
grunt
```

**Clean your project**
```
grunt clean
```

**Launch your server**
```
./start.sh
```

**Launch your server as production server**
```
sudo ./start.sh PROD
```

[starterkit]: https://github.com/paqman/starter-kit
[nodejs]: http://nodejs.org/
